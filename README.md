
Приложение представляет собой систему интернет-магазина, где основные сущности — пользователи, товары, заказы и их составляющие.
Приложение поддерживает авторизацию с помощью AuthX через заголовки.
Пользователи делятся по ролям на: гостей, авторизованных пользователей и администраторов.
Гость считается неавторизованным пользователем и имеет возможность просматривать товары и искать их.
При регистрации гость вводит данные о себе; если пароли совпадают и почта не занята, в базу заносится новый пользователь, а гость получает права авторизованного пользователя.

Авторизованный пользователь имеет возможность:
просматривать и искать все товары,
создавать и просматривать заказы для себя,
забирать доставленные заказы,
добавлять товары в заказы, если они ещё не доставлены,
смотреть и изменять свой профиль или удалить его.

Пользователь имеет доступ только к своему профилю, своим заказам и добавленным товарам.

Если пользователь уже зарегистрирован, но не авторизован, достаточно ввести почту и пароль; если они совпадают с данными в базе, пользователь авторизуется.

Для получения прав администратора необходимо зарегистрироваться под специально выделенной для этого почтой — admin@gmail.com.
Администратор имеет все права обычного пользователя и, сверх того, может:
просматривать все заказы, добавленные товары и профили всех зарегистрированных пользователей,
добавлять в базу товары,
изменять статус заказов,
менять информацию о продуктах,
удалять товары.

ВНИМАНИЕ в базе данных уже есть записи по всем таблицам, однако, зайти от лица такого пользователя не получится, так как они не проходили регистрацию,
ноебходимо зарегестрировать нового пользователя что бы получить доступ
При регистрации/логине приложение отправляет токен, который нужно вставить в swagger ui через кнопку authorize и тогда будут доступны права пользователя в зависимости от роли пользователя
Формат втсавки: Bearer "ТОКЕН"
Для получения прав админа нужно регистрироваться с почтой admin@gmail.com
